Declaración de Accesibilidad

Última actualización: Noviembre 2025

En OnSeguros creemos en una web para todos. Por eso, priorizamos el uso de tecnologías accesibles como etiquetas ARIA, navegación semántica y soporte para lectores de pantalla, asegurando que cualquier persona pueda utilizar nuestros servicios con comodidad y autonomía.

Implementamos soluciones prácticas para mejorar tu experiencia:

  • - Navegación por Teclado: Recorré todo el sitio usando Tab, flechas y Enter.
  • - Etiquetas ARIA: Brindamos contexto adicional para tecnologías asistivas.
  • - Imágenes Descriptivas: Uso de texto alternativo en elementos visuales.
  • - Adaptabilidad Visual: Soporte para alto contraste y modos claro/oscuro.
  • - HTML Semántico: Estructura lógica para una lectura clara por máquinas y humanos.
