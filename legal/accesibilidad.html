Declaración de Accesibilidad

Última actualización: Noviembre 2025

En OnSeguros nos comprometemos a garantizar que nuestro sitio web sea accesible para todas las personas, incluyendo aquellas con discapacidades. Trabajamos continuamente para mejorar la accesibilidad y usabilidad de nuestra web, cumpliendo con los estándares internacionales.

Nuestro sitio web se esfuerza por cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 nivel AA. Estas pautas ayudan a hacer el contenido web más accesible para personas con discapacidades, incluyendo:

  • - Discapacidades visuales (ceguera, baja visión, daltonismo)
  • - Discapacidades auditivas
  • - Discapacidades motoras
  • - Discapacidades cognitivas

Nuestro sitio web incluye las siguientes características de accesibilidad:

  • - Navegación por teclado: Todas las funcionalidades son accesibles mediante teclado
  • - Modo claro/oscuro: Selector de tema para ajustar el contraste según preferencias
  • - Texto alternativo: Todas las imágenes incluyen descripciones alternativas
  • - Estructura semántica: HTML semántico para facilitar la navegación con lectores de pantalla
  • - Etiquetas ARIA: Uso de atributos ARIA para mejorar la accesibilidad
  • - Contraste de colores: Colores con suficiente contraste para legibilidad
  • - Tamaños de texto: Fuentes legibles y escalables
  • - Formularios accesibles: Etiquetas claras y mensajes de error descriptivos

Para navegar nuestro sitio usando tecnologías asistivas:

  • - Usá la tecla Tab para moverte entre elementos interactivos
  • - Usá las teclas de flecha para navegar por menús y opciones
  • - Presioná Enter o Espacio para activar botones y enlaces
  • - Los lectores de pantalla pueden usar landmarks (role="banner", "main", "navigation", etc.)

Nuestro sitio web ha sido testeado con:

  • - Navegadores modernos (Chrome, Firefox, Safari, Edge)
  • - Lectores de pantalla (NVDA, JAWS, VoiceOver)
  • - Dispositivos móviles (iOS y Android)
  • - Navegación por teclado

A pesar de nuestros esfuerzos, algunas áreas del sitio pueden tener limitaciones de accesibilidad:

  • - Contenido de terceros (como mapas o videos embebidos) puede no ser completamente accesible
  • - Documentos PDF antiguos pueden no estar optimizados para accesibilidad

Estamos trabajando activamente para resolver estas limitaciones.
